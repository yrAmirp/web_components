# Lit WEB-COMPONENTS


## Установка

Установка зависимостей:

```bash
npm i
```

## Тестирование

```bash
npm test
```

## Запуск

```bash
npm run serve
```

## Пользовательский элемент 
Добавьте пользовательский элемент 
`<course-card> </course-card>`
для отображения карточки курса.

  > Пропишите атрибуты, как в обычном html-теге, для добавления информации



## Атрибуты

| Название | Атрибут | Значение по умолчанию |
| ----- | ----- | ----- |
| Размер карточки | size | s |
| Название курса | titleCourse | Курс... |
| Цена за занятие | price | ... |
| Количество занятий  | amountLessons | ... |
| Длительность занятий | durationLessons | ... |
| Особенности курса | features |  |
| Рейтинг курса | rating | 0.0 |
| Иконки | icons |  |

<details><summary>Доступные значения для icons</summary>
  
   - udemi
   - domZnanii
  
</details>

  > Значения можно прописывать через запятую
 
 
  > Особенности курса также прописываются через запятую
